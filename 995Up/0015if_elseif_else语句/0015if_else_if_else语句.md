## if(条件1)……else if(条件2)……else……语句

当需要多种互斥（独立）的分支时，可以使用if(条件1)……else if(条件2)……else……语句。

```java
if(条件1){
	//条件1 满足时执行这里
} else if(条件2){
	//条件2 满足时执行这里
} else if (条件3) {
……
……
} else {
	//哪个条件都不满足，执行这里
}
```
